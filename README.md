## 🚀 Installation in Kodi

Füge diese Quelle in Kodi unter **Dateimanager → Quelle hinzufügen** ein:

```
https://mr-evil1.github.io/kodi-expanded-repo/
```

Anschließend kannst du das Repository über **Addons → Aus ZIP-Datei installieren** einbinden und die enthaltenen Addons installieren.

---

## 📦 Enthaltene Addons

- **[GEZ] KiNO** – Filme ab 78 Minuten Laufzeit
- **Bild.de News** – Aktuelle Nachrichten von Bild.de
- **You Don't Know Jack - Fan Quiz** – Fan-Quiz mit Highscore-System
- **RTL+** – Serien, Shows, Filme und Live-TV
- **Wer wird Millionär?** – Die bekannte Quizshow zum Mitspielen
- **Downloader ATV** – Browserbasierter Download-Manager
- **httpx** – HTTP-Client-Modul für Python 3

Die vollständige Liste mit Beschreibungen, Fanart und direkten Download-Links findest du in der [Addon-Übersicht](https://mr-evil1.github.io/kodi-expanded-repo/index.html).
